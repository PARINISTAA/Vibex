# You to Us Frontend

## How to run

1. Install Node.js and Expo CLI globally if you haven't:
   ```bash
   npm install -g expo-cli
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Start the app:
   ```bash
   expo start
   ```

4. Open on your device with Expo Go or run on web with `expo start --web`
